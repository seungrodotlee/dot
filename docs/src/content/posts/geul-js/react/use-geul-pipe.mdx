# use-geul-pipe

import UseGeulPipeExample from "../../../../domains/examples/use-geul-pipe.example.component";

`use-geul-pipe` is hook for run typing effect with multiple values in order.

```ts
function useGeulPipe(
  values: string[],
  options: UseGeulPipeOptions,
): UseGeulPipeReturn;
```

## `values`

Values that will be sequential results of the typing effect.

## `UseGeulPipeOptions`

<table>
  <thead>
    <tr>
      <th>option</th>
      <th>type</th>
      <th>description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>`speed`</td>
      <td>`number`</td>
      <td>Speed of typing effect (ms)</td>
    </tr>
    <tr>
      <td>`decomposeOnBackspace`</td>
      <td>`boolean | undefined`</td>
      <td>
        Whether to have the effect of separating consonants when backspace
      </td>
    </tr>
  </tbody>
</table>

## `UseGeulPipeReturn`

<table>
  <thead>
    <tr>
      <th>member</th>
      <th>type</th>
      <th>description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>`geul`</td>
      <td>`string`</td>
      <td>Typed value</td>
    </tr>
    <tr>
      <td>`isRunning`</td>
      <td>`boolean`</td>
      <td>Running status of the `geul` instance</td>
    </tr>
    <tr>
      <td>`next`</td>
      <td>`() => void`</td>
      <td>funtion that start next typing effect</td>
    </tr>
    <tr>
      <td>`reset`</td>
      <td>`() => void`</td>
      <td>funtion that reset typing effect</td>
    </tr>
  </tbody>
</table>
