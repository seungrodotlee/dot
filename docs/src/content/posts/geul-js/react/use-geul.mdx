# use-geul

import UseGeulExample from "../../../../domains/examples/use-geul.example.component";

`use-geul` is basic hook for run typing effect.

```ts
function useGeul(value: string, options: UseGeulOptions): UseGeulReturn;
```

## `value`

Value that will be result of typing effect.

## `UseGeulOptions`

<table>
  <thead>
    <tr>
      <th>option</th>
      <th>type</th>
      <th>description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>`speed`</td>
      <td>`number`</td>
      <td>Speed of typing effect (ms)</td>
    </tr>
    <tr>
      <td>`initial`</td>
      <td>`string | undefined`</td>
      <td>Initial value that displayed before typing</td>
    </tr>
    <tr>
      <td>`decomposeOnBackspace`</td>
      <td>`boolean | undefined`</td>
      <td>
        Whether to have the effect of separating consonants when backspace
      </td>
    </tr>
  </tbody>
</table>

## `UseGeulReturn`

<table>
  <thead>
    <tr>
      <th>member</th>
      <th>type</th>
      <th>description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>`geul`</td>
      <td>`string`</td>
      <td>Typed value</td>
    </tr>
    <tr>
      <td>`isRunning`</td>
      <td>`boolean`</td>
      <td>Running status of the `geul` instance</td>
    </tr>
    <tr>
      <td>`run`</td>
      <td>`(onTypeEnd?: (() => void)) => void`</td>
      <td>
        funtion that start typing effect
      </td>
    </tr>
    <tr>
      <td>`reset`</td>
      <td>`() => void`</td>
      <td>
        funtion that reset typing effect
      </td>
    </tr>
  </tbody>
</table>
