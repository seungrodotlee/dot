# Install Geul.js for React

Install via Package Manager

```sh
  npm i @d0t/react-geul
```

# Usage

You can use `Geul.js` by import `@d0t/react-geul` package and just simply call hooks. `@d0t/react-geul` provides three hooks.

- `useGeul`
- `useDynamicGeul`
- `useGeulPipe`
